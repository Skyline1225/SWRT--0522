# 文档内容更新工具 - 打包说明

## 快速开始

### 方法一：一键打包（推荐）

1. 双击运行 `打包.bat` 文件
2. 等待打包完成
3. 打包成功后，可执行文件在 `dist` 文件夹中：`文档内容更新工具.exe`

### 方法二：手动打包

```bash
# 1. 安装 PyInstaller
pip install pyinstaller

# 2. 进入项目目录
cd "f:\AI\代码\SWRT文档打点\0503-test\doc_updater"

# 3. 执行打包
pyinstaller --clean package.spec
```

## 打包后的文件

打包成功后，你会得到：

```
dist/
└── 文档内容更新工具.exe    <-- 这就是可执行文件
```

## 分享给其他人

直接将 `dist/文档内容更新工具.exe` 文件发送给其他 Windows 用户即可，他们双击就能运行，不需要安装 Python 或任何依赖。

## 注意事项

1. **杀毒软件拦截**：第一次运行时，部分杀毒软件可能会误报，选择"允许运行"即可
2. **文件大小**：打包后的文件约 50-100MB，这是正常的，因为包含了所有依赖库
3. **Windows版本**：需要 Windows 7 或更高版本
4. **首次运行**：首次打开可能需要几秒钟加载

## 常见问题

**Q: 打包失败怎么办？**
A: 确保已安装所有依赖：`pip install -r requirements.txt`

**Q: 打包后的软件闪退？**
A: 修改 package.spec 中的 `console=False` 改为 `console=True`，重新打包查看错误信息

**Q: 如何给软件加图标？**
A: 准备一个 .ico 图标文件，修改 package.spec 中的 `icon=None` 为 `icon='你的图标.ico'`
